#01_SkyMachine2#
****************************************************************************************************************************************************

・作品名		SkyMachine2
・ジャンル		2Dシューティングゲーム
・製作人数		1人
・製作期間		7月下旬～(9月下旬完成予定)
・開発環境		C++,DirectX9
・使用ツール	VisualStudio2015, AdobePhotoshop, GitHub

SkyMachineの続編という体で制作を始めましたが、今作は「ダライアス外伝」を強く意識しています。
ステージ構成や演出、背景のアニメーションに注力し、ユーザー側が飽きないよう隅々まで細かな演出を入れるように注意して制作しています。

#02_MemoryHit#
****************************************************************************************************************************************************

・作品名		Memory Hit ～覚えろ!!そして探せ!!～
・ジャンル		記憶パズルゲーム
・製作人数		4人
・製作期間		10時間(3月下旬)
・開発環境		C言語,DirectX9
・使用ツール	VisualStudio2015, AdobePhotoshop, GitHub

当日発表のお題に沿って10時間でゲームを作る、『ハッカソン』イベントにて4人チームで制作しました。
お題は『らいと』。40分程度で企画をまとめ、最終的に画面の暗転を利用した記憶ゲームになりました。
開発ではメインプログラマーを担当しました。(アイテム配置、得点処理、プログラムのマージ等)
